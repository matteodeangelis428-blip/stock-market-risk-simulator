# Stock Market Risk Simulator

This project models possible future stock market outcomes using a Monte Carlo simulation based on historical S&P 500 data.

The simulation generates thousands of possible market paths over a 10-year period to analyze investment risk and uncertainty.

## Method
1. Historical market returns are analyzed
2. Average return and volatility are calculated
3. A Monte Carlo simulation generates 1000 possible future market paths

## Purpose
This project demonstrates how statistical modeling can be used to analyze financial risk. Techniques like these are commonly used in actuarial science and quantitative finance.
